# vue-lint-demo

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### For ESLint's and Prettier lint check,

```
yarn lint
```

### For ESLint's and Prettier lint fix,

```
yarn lint-fix
```
